Launging External Tools

Commands listed under the separator "----" in the command menu are for launching externals tools.

To control and manage detail of the repository, choose "Terminal" menu which open a terminal window of which working directory is the contents of the bundle.

GitHub Deskop is recommend as a GUI client to view git repository of script bundles, because only GitHub Desktop can display differences of script files (.scpt) along "git config diff.osascript.textconv osadecompile" setting.

Other GUI clients e.g. SourceTree, GitKraken GitUp ignore "git config diff.osascript.textconv osadecompile" setting and treat script files as binary.
