git-scptd とは？

スクリプトバンドル、AppleScript アプレット/ドロップレットを git でバージョン管理します。

必要なシステム

OS の動作条件、および必要なソフトウェア

バージョン管理を開始する

スクリプトエディタで開いているスクリプトバンドル/アプリケーションに git レポジトリをセットアップします。

最初のコミット

リストボックスから様々な git の操作を選択できます。

外部ツールの起動

ターミナルと GitHub Desktop でレポジトリを開けます。

設定の更新

既存のレポジトリの設定を行えます。
