git-scptd とは？

スクリプトバンドルやAppleScript アプレット/ドロップレットなど、AppleScript エディタで保存されるバンドルを、git でバージョン管理するツールです。

  1. バンドル内にレボジトリを作ります。
  2. バンドル内のスクリプトファイル（.scpt ファイル）の差分を git で見れるようにレポジトリを設定します。
  3. よく使うコマンドのメニューを提供します。
  4. 外部ツールの起動を行うリストメニューを提供します。
