バージョン管理を開始する

まずは Script エディタでスクリプトバンドルを開いてください。そして、git-scptd をスクリプトメニューから起動します。git-scptd は起動時に、最前面にある書類を対象とするので、なんらかのランチャーを使って実行してください。僕は、LaunchBar を使っています。

すると、まず最初に git init を実行して repository を作るか聞いてきます。

上記のダイアログで OK ボタンを押すと、git init を実行して、repository の作成だけではなく、

  • バンドル内の Contents を add する。
  • .scpt ファイルの diff を実行する際に使うフィルターの設定。
    • .git/info/attributes のファイルの設定。
    • git config diff.osascript.textconv osadecompile を実行。
  • .git/hooks/.pre-commit の設定。
    • .scpt ファイルの diff をとって、変更が無い場合は、.scpt のコミットを抑制する。

を行います。
