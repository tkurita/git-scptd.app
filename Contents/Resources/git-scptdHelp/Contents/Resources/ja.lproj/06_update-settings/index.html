設定の更新

Update Settings は、レボジトリに対する設定、すなわち、

  • .scpt ファイルの diff を実行する際に使うフィルターの設定。
    • .git/info/attributes のファイルの設定。
    • git config diff.osascript.textconv osadecompile を実行。
  • .git/hooks/.pre-commit の設定。
    • .scpt ファイルの diff をとって、変更が無い場合は、.scpt のコミットを抑制する。
  • .gitignore
    • main.recover.rtf, Script Debugger.plist, Script Debugger.list が登録されます。

をやり直します。何らかの理由で、スクリプトバンドル用の設定が失われた場合、将来、git-scptd がアップデートされた場合に使ってください。
